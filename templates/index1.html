A Website Creation for Media Don Entertainment

Home      About us          Contact Us      User Registration           Login

The main aim of the project is develop web application for Media Don entertainment Company. This proposed application helps to the company they can maintain all the workflow through the website. In the organization previously management keep all the process into separate record book. New customer register, and design selection and order details, production details and delivery details all are maintained by individual record book and needed more work time process for searching any required data. To avoid these all problems this website is developed for the organization. Using this website admin can promote their business to the society peoples.Admin can able to upload complete event related information event may be marriage shoot ,wedding shoot ,catering service, decoration etc along with package information and photo gallery. This project the user has to register to the website. After the registration user will get login id and password. The login id and password helps to user login the website. The user can view complete service and package details of Media Don entertainment Company after login. And also user can book their service efficient way using this website. After making booking process Media Don entertainment Company admin will see the booking request they will give confirmation to client effectively using this website.. >
